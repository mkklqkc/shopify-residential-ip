## 什么是"Shopify美国固定IP"，跟普通代理有什么区别

**美国固定IP（Static Residential IP）** 是指由美国本地互联网运营商（ISP）直接分配给家庭用户的公网地址，长期固定不变，不会因为重连或时间推移而改变。

这跟两类常见产品都不一样：

- **动态代理IP**：每次请求可能走不同出口，IP 会变，适合爬取数据，不适合账号登录
- **数据中心IP（机房IP）**：来自商业服务器机房，价格便宜，但容易被平台识别为非真实用户

固定的美国住宅 IP，平台看到的是一个长期稳定的普通美国家庭网络。这对需要保持账号稳定、降低风控触发率的 Shopify 卖家来说，才是真正有用的东西。

## 市面上"住宅IP"有多少是假的

这里有个坑，值得提一下。

市面上打着"住宅IP"旗号的产品，大致分三类：

- **真正的家庭宽带IP**：服务商在美国本地拉真实家庭宽带，IP 来自 AT&T、Comcast 这类运营商，WHOIS 查出来就是住宅地址
- **双ISP IP**：部分机房 IP 会显示两个 ISP 信息，看起来像住宅，但实际上并不是真正从家庭地址出来的
- **代理池轮转IP**：从住宅用户借用出口，共享、短暂，每次 IP 都不同

对于 Shopify 运营来说，能用的只有第一种——长期固定、来源真实的家庭宽带地址。其他两种要么不够稳定，要么早被平台标记。

**Q：Shopify 一定需要美国固定IP吗？**
